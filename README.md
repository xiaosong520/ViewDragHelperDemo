### 主要的功能点有如下几个：
1. 滑动方向判定。
2. 单个方向的拖拽限制。
3. 事件分发以及拦截。
4. 平移动画处理。
5. 下拉时缩放及背景透明处理。
6. 背景高斯图片替换处理。
7. 嵌套ScrollView / RecyclerView事件冲突处理。
8. 多点触控 Invalid pointerId 问题解决。
